## 关于
### simple-web4j
    'simple-web4j'一个基于java语言的简易Web框架。
### 起源
    因为工作中一直依赖spring相关的组件进行开发。渐渐的便好奇它为我们的做了些什么，于是开始看源码，看的过程中萌生出了一个想法，自己来实现一个类似的框架，加深自己的理解，同时也可以说是一个源码阅读笔记。
